# LifeTwin -- Digital Life Simulator

AI-powered life dashboard that simulates health, finance, career, and wellness trajectories.

## Tech Stack

- **Framework**: Next.js 16 (App Router, TypeScript)
- **Styling**: Tailwind CSS 4
- **State**: Zustand
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- **Health Dashboard**: Track sleep, exercise, nutrition, stress, BMI, and heart rate
- **Finance Dashboard**: Net worth projections, expense breakdowns, investment portfolio
- **Career Dashboard**: Skills radar, satisfaction tracking, career level progression
- **Wellness Dashboard**: Happiness gauges, mood calendar, social connections
- **Simulation Engine**: 5 built-in life scenarios with realistic modeling
- **Timeline View**: Multi-dimension overlay with life event markers

## Docker

```bash
docker-compose up --build
```
